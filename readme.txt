Any Image can be croped in square by user defined
1. put cropper folder in project asset directory
2. put the code in view.html to project template file
3. put the code in controller.php to project controller file
4. and finally put the javascript function to project javascript section

put cropper plugin Link:
<script src="/cropper/cropper.css'"></script>
<script src="/cropper/cropper.min.js'"></script>
